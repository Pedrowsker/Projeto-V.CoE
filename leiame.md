# Sistema de Auditoria Inteligente de Fretes e Notas Fiscais

## 📌 Sobre o Projeto
Este projeto simula um fluxo de automação para um Centro de Excelência (CoE). O objetivo é transformar dados brutos de filiais em inteligência de negócio.

## 🛠️ Tecnologias
- **Python**: Limpeza de dados (Pandas).
- **SQL**: Extração de dados.
- **Power BI**: Dashboards.
- **Excel/VBA**: Automação legada.
